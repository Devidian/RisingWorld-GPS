G P S  -  A Rising World Java plug-in for navigation and self-location

Installation

Expand the ZIP file into the  plugins  folder of the main Rising World
folder, obtaining a folder structure like:

── <RisingWorld>
    ├── plugins
    │    ├── gps
    │    │    ├── assets/
    │    │    ├── locale/
    │    │    ├── COPYING
    │    │    ├── gps.jar
    │    │    ├── readme.txt
    │    │    └── settings.properties


Configuration

The  settings.properties   file contains the configuration settings
which can be customised. Each setting is commented: refer to the file
itself for details.
